# ISW 工作坊议程模板

> 本模板用于生成完整的 ISW 医学院教师培训工作坊议程。标准 ISW 工作坊为3-4天集中培训，不少于24学时。

---

## 工作坊基本信息

| 项目 | 内容 |
|------|------|
| **工作坊名称** | 【如：XX医学院ISW教学技能工作坊】 |
| **培训对象** | 【如：新任临床教师/护理学骨干教师/基础医学青年教师】 |
| **培训人数** | 【每组4-6人，共X组，总计X人】 |
| **引导员** | 【引导员姓名及FDW认证编号】 |
| **培训时间** | 【YYYY-MM-DD 至 YYYY-MM-DD】 |
| **培训地点** | 【场地信息】 |
