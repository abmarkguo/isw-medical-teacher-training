## 教师角色时间线

| 时间节点 | 教师角色 | 关键行为 |
|---------|---------|---------|
| 0:00-0:03 | 引导者 | 发布导入活动指令，营造氛围 |
| 0:03-0:05 | 讲述者 | 呈现学习目标 |
| 0:05-0:09 | 观察者 | 前测活动，记录基线 |
| 0:09-0:32 | 引导者 | 参与式学习，巡视+适时介入 |
| 0:32-0:39 | 评价者 | 后测活动，对照前测 |
| 0:39-0:45 | 促导者 | 总结活动，引导反思 |

## 质量自检清单

### 全员参与保障
- [ ] 是否设计了个人思考环节？
- [ ] 小组人数是否合理（3-6人）？
- [ ] 是否有个人产出要求？
- [ ] 角色是否轮换？

### 认知深度保障
- [ ] 互动活动对应目标Bloom层次？
- [ ] 问题是否开放？
- [ ] 是否要求提供理由/证据？
- [ ] 是否有追问机制？

### 可操作性保障
- [ ] 活动指令是否清晰？
- [ ] 时间分配是否合理？
- [ ] 材料是否易准备？
- [ ] 是否有应急预案？

### 可评价性保障
- [ ] 互动产出可收集？
- [ ] 有评价标准？
- [ ] 教师能观察关键行为？
- [ ] 有记录方式？

### 医学教育特色
- [ ] 互动中保持医疗专业场景真实感？
- [ ] 角色扮演后有debriefing环节？
- [ ] 案例保护患者隐私？
- [ ] 互动中融入伦理思考？
