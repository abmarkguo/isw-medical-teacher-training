# BOPPPS 教学设计模板

> 本模板用于生成完整的 BOPPPS 医学教学教案。使用时将【】中的内容替换为实际教学设计。

---

## 基本信息

| 项目 | 内容 |
|------|------|
| **课程名称** | 【如：内科学/病理学/基础护理学】 |
| **教学主题** | 【如：急性心肌梗死的诊断与处理】 |
| **授课对象** | 【如：临床医学本科三年级/护理学本科二年级/住培学员第一年】 |
| **课时安排** | 【如：45分钟/90分钟/10分钟微格教学】 |
| **学科类别** | 【基础医学/临床医学/护理学/药学/跨学科】 |
| **授课教师** | 【教师姓名】 |
| **授课日期** | 【YYYY-MM-DD】 |

## 学习目标

### 知识目标
1. 【能说出/描述/解释……】
2. 【能归纳/比较/分析……】

### 技能目标
1. 【能实施/操作/执行……】
2. 【能在模拟/真实情境中完成……】

### 态度/人文目标
1. 【能体认到/表现出……】
2. 【能在……情境中体现人文关怀/伦理意识】

> 目标编写检查：
> - [ ] 使用了可观测的行为动词？
> - [ ] 包含知识/技能/态度三个维度？
> - [ ] 数量控制在3-5个？
> - [ ] 符合ABCD原则（对象+行为+条件+标准）？

## BOPPPS 教学流程

### B — Bridge-in 导入（【X】分钟）

| 要素 | 设计 |
|------|------|
| **导入策略** | 【临床案例导入/问题情境导入/数据冲击导入/实物标本导入/角色扮演导入/新闻事件导入】 |
| **具体内容** | 【详细描述导入内容，如案例摘要、数据、问题等】 |
| **与目标关联** | 【说明导入如何连接学习目标】 |

### O — Objective 目标（【X】分钟）

| 要素 | 设计 |
|------|------|
| **呈现方式** | 【口头说明/PPT展示/学案印发/学生朗读】 |
| **目标说明** | 【简明呈现3-5个学习目标】 |

### P — Pre-assessment 前测（【X】分钟）

| 要素 | 设计 |
|------|------|
| **前测方法** | 【MCQ投票/概念图填空/病例判断/自评量表/头脑风暴】 |
| **前测内容** | 【具体题目或任务】 |
| **结果利用** | 【如何根据前测结果调整教学】 |

### P — Participatory Learning 参与式学习（【X】分钟）

| 活动编号 | 活动名称 | 活动类型 | 对应目标 | 时长 | 详细操作 | 材料/工具 |
|:---:|---------|---------|---------|:---:|---------|---------|
| 1 | 【活动名】 | 【个人/小组/全班】 | 【目标X】 | 【X分】 | 【详细操作步骤】 | 【所需材料】 |
| 2 | 【活动名】 | 【个人/小组/全班】 | 【目标X】 | 【X分】 | 【详细操作步骤】 | 【所需材料】 |
| 3 | 【活动名】 | 【个人/小组/全班】 | 【目标X】 | 【X分】 | 【详细操作步骤】 | 【所需材料】 |

> 参与式学习检查：
> - [ ] 学生活动时间 > 教师讲授时间？
> - [ ] 活动类型多样（非全部"讨论"）？
> - [ ] 活动与目标层次匹配？
> - [ ] 整合了医学教学法（PBL/CBL/TBL/模拟）？

### P — Post-assessment 后测（【X】分钟）

| 要素 | 设计 |
|------|------|
| **后测方法** | 【MCQ/病例分析/操作观察/角色扮演评分/模拟场景/反思问卷】 |
| **后测内容** | 【具体题目或任务】 |
| **与前测对照** | 【说明后测如何与前测形成对照（同维度不同题）】 |
| **即时反馈** | 【如何对后测结果进行即时反馈】 |
| **临床考核对接** | 【如对接OSCE/mini-CEX/DOPS，说明对接方式】 |

### S — Summary 总结（【X】分钟）

| 要素 | 设计 |
|------|------|
| **总结策略** | 【一分钟论文/电梯演讲/临床应用链接/思维导图/承诺行动】 |
| **具体内容** | 【详细描述总结活动】 |
| **课后延伸** | 【布置课后任务/链接下次课程/临床实践建议】 |
