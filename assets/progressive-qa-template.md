### 追问策略库

**当学生回答不完整时：**
- 澄清追问："{你能具体说说……的意思吗？}"
- 补充追问："{还有其他可能的……吗？}"

**当学生回答错误时：**
- 证据追问："{你做出这个判断的依据是什么？}"
- 反例追问："{如果患者的情况是XX，你的结论还成立吗？}"
- 重定向追问："{我们再看看这些信息，有没有遗漏什么？}"

**当学生回答正确但浅层时：**
- 深化追问："{为什么这很重要？}"
- 迁移追问："{这和之前学过的XX有什么联系？}"
- 评价追问："{在临床实践中，这个原则有什么局限性？}"

### 问题链逻辑图

```
Q1（信息提取）
  ↓ 桥接语："好，我们知道了这些信息，那么……"
Q2（模式识别）
  ↓ 桥接语："很好，你提到了XX，那我们怎么区分？"
Q3（鉴别推理）
  ↓ 桥接语："所以需要进一步检查，选什么？"
Q4（检查选择+结果解读）
  ↓ 桥接语："结果出来了，现在能确诊了吗？"
Q5（决策制定）
  ↓ 桥接语："治疗方案确定了，但如果情况改变呢？"
Q6（反思评价）
```

## 互动方式速查

| 互动方式 | 适用Bloom层 | 操作方法 | 时间 |
|---------|:-----------:|---------|:----:|
| 个人思考 | L1-L6 | 独立思考不交流 | 5s-1min |
| Think-Pair-Share | L2-L5 | 思考30s→同桌讨论1min→分享 | 2-3min |
| 小组讨论 | L3-L6 | 3-5人讨论后汇报 | 3-10min |
| 全班投票 | L1-L3 | 选项投票/扫码 | 1-2min |
| 角色扮演 | L5-L6 | 模拟场景表演 | 5-10min |
| 白板展示 | L4-L6 | 小组在白板画推理过程 | 3-5min |

## 质量自检清单

- [ ] 问题是否由浅入深、层层递进？
- [ ] 相邻问题是否有逻辑关联（非简单并列）？
- [ ] 是否覆盖了目标的Bloom层次？
- [ ] 每个问题是否标注了互动方式和等待时间？
- [ ] 是否为每个问题准备了追问策略？
- [ ] 桥接语是否自然流畅？
- [ ] 问题是否与临床情境绑定？
- [ ] 最后一个问题是否指向高阶思维？
- [ ] 总时长是否在预算范围内？
- [ ] 是否有应急问题（备选）？
