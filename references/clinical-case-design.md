# 详细临床案例编写指南

## 一、案例设计原则

### 核心价值
详细临床案例是医学教育中最具整合性的教学工具，一个优质案例可以同时承载知识建构、临床推理训练、医学人文熏陶和跨学科链接四重功能。

### 五项设计原则

1. **真实性（Authenticity）**
   - 基于真实临床病例改编，保留核心诊疗逻辑
   - 患者信息严格脱敏（姓名、住院号、住址、就诊日期等）
   - 病史采集、体征描述、检查结果符合临床实际
   - 避免教科书式的"完美病例"，保留临床不确定性

2. **教学性（Instructional Alignment）**
   - 案例与BOPPPS学习目标直接对齐
   - 每个案例信息点都有明确的教学用途
   - 案例难度匹配授课对象层次
   - 案例可覆盖多个学习目标（知识+技能+态度）

3. **渐进性（Progressive Disclosure）**
   - 信息分阶段释放，模拟临床真实诊疗过程
   - 每阶段信息量适当，避免信息过载
   - 前阶段信息为后阶段推理提供基础
   - 关键信息可设置"干扰项"训练鉴别能力

4. **整合性（Integration）**
   - 融合基础医学（解剖/生理/病理/药理）与临床医学
   - 嵌入医学人文/伦理/法律要素
   - 鼓励跨学科视角（如护理、药学、康复）
   - 链接预防医学和公共卫生维度

5. **伦理维度（Ethical Dimension）**
   - 案例中自然嵌入伦理两难或人文关怀情境
   - 不回避临床实践中的价值冲突
   - 提供伦理讨论的空间和材料
   - 尊重患者尊严，避免污名化表述

## 二、案例结构标准

### 完整案例包含以下模块：

```
案例概要卡 → 教学目标映射 → 患者基本信息 → 就诊情境
→ 第一阶段：主诉与现病史 → 思考问题
→ 第二阶段：既往史与个人史 → 思考问题
→ 第三阶段：体格检查 → 思考问题
→ 第四阶段：辅助检查 → 思考问题
→ 第五阶段：诊断与鉴别诊断 → 思考问题
→ 第六阶段：治疗方案 → 思考问题
→ 第七阶段：随访与转归 → 思考问题
→ 伦理与人文讨论点
→ 跨学科链接
→ 教学使用说明
→ 参考答案与教学要点
```

### 各模块编写规范

#### 1. 案例概要卡
| 字段 | 说明 |
|------|------|
| 案例标题 | 简明扼要，体现核心诊断或教学主题 |
| 案例编号 | CASE-学科缩写-序号 |
| 适用学科 | 基础医学/临床医学/护理学/药学/跨学科 |
| 适用对象 | 本科X年级/住培X年/继续教育 |
| 教学时长 | 30分钟/45分钟/90分钟 |
| 教学法建议 | CBL/PBL/TBL/模拟教学 |
| 核心知识点 | 3-5个核心知识或技能 |
| Bloom层次 | 主要覆盖的Bloom层次 |
| 人文要素 | 案例涉及的医学人文主题 |

#### 2. 教学目标映射
将案例每个阶段与BOPPPS学习目标对应：
- 阶段1对应目标1（知识层）
- 阶段3对应目标2（应用层）
- 阶段5对应目标3（分析层）
- 阶段7对应目标4（评价/创造层）

#### 3. 患者基本信息
```
姓名：张某（化名）
性别：男
年龄：58岁
职业：出租车司机
民族：汉族
婚姻状况：已婚
就诊方式：120急救车送入
就诊时间：20XX年X月X日 X时
```

#### 4. 就诊情境
用叙事方式描述患者就诊场景（2-3段），包括：
- 来诊方式和环境
- 患者状态（意识、表情、体位）
- 陪同人员及情绪
- 首诊医护的第一印象
- 场景中的感官细节（声音、气味等）

#### 5-11. 七个阶段信息卡

每个阶段包含：
- **临床信息**：按真实病历格式编写
- **信息释放说明**：教师在何时、以何种方式呈现这些信息
- **思考问题**：2-3个引导问题（标注Bloom层次）
- **教学要点**：本阶段的教学重点和常见误区

#### 12. 伦理与人文讨论点
- 识别案例中的伦理两难
- 设计2-3个伦理讨论问题
- 提供伦理分析框架（四原则法/叙事伦理）
- 链接职业素养要素

#### 13. 跨学科链接
| 学科 | 链接点 | 深度 |
|------|--------|------|
| 解剖学 | 冠状动脉解剖与梗死部位对应 | 复习 |
| 生理学 | 心肌电生理与心律失常 | 理解 |
| 药理学 | 抗血小板药物作用机制 | 应用 |
| 预防医学 | 心血管疾病一级/二级预防 | 评价 |

#### 14. 教学使用说明
- 推荐教学法（CBL/PBL/TBL/模拟）
- 分组建议（人数/角色分配）
- 时间分配（各阶段建议时长）
- 材料准备（道具/设备/资料）
- 注意事项（敏感话题处理/时间控制）

#### 15. 参考答案与教学要点
- 每个思考问题的参考答案（非唯一标准答案）
- 常见错误回答及应对策略
- 延伸阅读推荐
- 最新指南要点提示

## 三、案例难度分级

| 等级 | 特征 | 适用对象 | 信息复杂度 | 干扰项 |
|------|------|---------|-----------|--------|
| L1入门 | 典型表现，单一诊断 | 本科低年级 | 低（信息明确） | 无 |
| L2基础 | 典型+1个合并症 | 本科中年级 | 中 | 1个 |
| L3进阶 | 非典型表现，需鉴别 | 本科高年级/实习 | 中高 | 2-3个 |
| L4高级 | 多系统受累，多学科 | 住培/专培 | 高 | 3个+ |
| L5复杂 | 不确定性高，伦理两难 | 主治/继续教育 | 极高 | 多个+信息缺失 |

## 四、不同学科案例编写要点

### 基础医学案例
- **链接策略**：从临床现象回溯基础机制
- **信息侧重**：病理改变、发病机制、分子基础
- **特色**：设置"机制探查"问题链
- **案例来源**：真实病例+实验室数据+病理图片

### 临床医学案例
- **链接策略**：完整的临床推理过程
- **信息侧重**：症状体征、检查结果、诊疗决策
- **特色**：信息渐进释放，模拟接诊过程
- **案例来源**：真实病例改编，保留不确定性

### 护理学案例
- **链接策略**：护理评估→护理诊断→护理措施→评价
- **信息侧重**：生命体征、护理观察、患者反应
- **特色**：突出护理评估和人文关怀
- **案例来源**：真实护理场景+护理记录

### 药学案例
- **链接策略**：处方分析→药物选择→用药监测→患者教育
- **信息侧重**：用药史、药物相互作用、不良反应
- **特色**：处方合理性分析和用药安全
- **案例来源**：真实处方+药学监护记录

## 五、案例质量控制清单

- [ ] 患者信息是否已脱敏？
- [ ] 临床信息是否符合医学准确性？
- [ ] 案例是否与学习目标直接对齐？
- [ ] 信息是否分阶段渐进释放？
- [ ] 每个阶段是否有思考问题？
- [ ] 思考问题是否覆盖多个Bloom层次？
- [ ] 是否嵌入了伦理/人文讨论点？
- [ ] 是否有跨学科链接？
- [ ] 参考答案是否标注"非唯一标准"？
- [ ] 是否提示参照最新临床指南？
- [ ] 案例难度是否匹配授课对象？
- [ ] 是否有干扰项训练鉴别能力？
- [ ] 教学使用说明是否清晰可操作？

## 六、案例与BOPPPS的整合

| BOPPPS环节 | 案例使用方式 |
|-----------|-------------|
| Bridge-in | 案例初始场景作为情境导入 |
| Objective | 案例教学目标映射说明 |
| Pre-assessment | 案例第一阶段作为前测（快速判断） |
| Participatory Learning | 案例核心阶段展开，分组讨论+渐进信息 |
| Post-assessment | 案例变式（改变关键变量）作为后测 |
| Summary | 案例完整诊疗路径回顾+伦理反思 |
