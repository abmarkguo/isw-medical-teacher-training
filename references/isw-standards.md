# ISW 国际与国内标准参考

## 一、ISW 国际标准

### 1.1 ISW 的起源与定位

ISW（Instructional Skills Workshop，教学技能工作坊）起源于1978年加拿大英属哥伦比亚省（British Columbia），由 Douglas Kerr 等人开发。它是一个**同行驱动、体验式、非评价性**的教师教学发展项目，至今在全球30多个国家和地区广泛应用。

**核心特征：**
- 以 BOPPPS 模型为教学设计框架
- 以微格教学（Micro-teaching）为核心训练方式
- 以同行反馈（Peer Feedback）为主要成长机制
- 引导员（Facilitator）经 FDW（Facilitator Development Workshop）认证
- 强调非评价性、建设性、学习者中心的反馈文化

### 1.2 ISW 国际网络标准

ISW International Network（ISWIN）规定的核心标准：

| 要素 | 标准要求 |
|------|---------|
| **培训时长** | 不少于24小时（通常3-4天集中培训） |
| **小组规模** | 每组4-6名参训者 + 1名认证引导员 |
| **微格教学轮次** | 每人至少完成3轮微格教学 |
| **每轮时长** | 10分钟教学 + 10-15分钟反馈 |
| **教学设计** | 必须使用 BOPPPS 模型 |
| **反馈方式** | 结构化同行反馈（非评价性） |
| **引导员资质** | 须完成 FDW 认证培训 |
| **参与要求** | 全程参与，不得缺席超过10% |

### 1.3 ISW 三级认证体系

```
ISW（教学技能工作坊）
  └── FDW（引导员发展工作坊）
       └── TDW（培训师发展工作坊）
```

| 层级 | 目标人群 | 核心能力 |
|------|---------|---------|
| **ISW** | 一线教师 | 掌握 BOPPPS 模型，能设计并实施有效教学 |
| **FDW** | ISW 结业教师 | 能引导 ISW 工作坊，提供建设性反馈 |
| **TDW** | 资深引导员 | 能培训新的 FDW 引导员 |

## 二、ISW 国内标准与医学教育对接

### 2.1 中国高校 ISW 推广情况

ISW 自2010年代引入中国后，在医学院校中推广迅速。主要推动力量：
- 各省师资培训中心
- 中国医学教育师资发展联盟
- 教育部医学教育认证体系对接需求

### 2.2 医学院校教师教学发展标准对接

**《医学院校教师教学发展指南》核心维度：**

1. **教学设计能力** — 与 BOPPPS 的 O/Pre-assessment 对接
2. **教学方法运用** — 与 BOPPPS 的 Participatory Learning 对接
3. **教学评价能力** — 与 BOPPPS 的 Pre/Post-assessment 对接
4. **教学反思能力** — 与 ISW 的反馈文化对接
5. **课程开发能力** — BOPPPS 的扩展应用

### 2.3 医学教育认证标准对接

**教育部临床医学专业认证标准（2023版）相关条目：**
- 教学方法：鼓励采用以学生为中心、以问题为导向的教学方法
- 教师发展：院校应建立教师教学发展机制
- 教学评价：建立形成性评价与终结性评价相结合的评价体系

**住培教学规范对接：**
- BOPPPS 可直接应用于教学查房设计
- 微格教学与 mini-CEX 评估理念相通
- 反馈文化与住培形成性反馈一致

### 2.4 医学院校 ISW 培训的特有要求

| 维度 | 通用 ISW | 医学院校特有要求 |
|------|---------|----------------|
| **教学情境** | 通用课堂 | 临床教学（病房、门诊、手术室、模拟实验室） |
| **学科特点** | 无特定学科 | 基础医学、临床医学、护理学、药学等差异化设计 |
| **教学法** | 通用参与式学习 | PBL、CBL、TBL、模拟教学、床旁教学 |
| **评价工具** | 通用前后测 | OSCE、mini-CEX、DOPS 等临床评价工具 |
| **人文要素** | 通用 | 医学伦理、医患沟通、职业素养、生命教育 |
| **安全意识** | 无特殊 | 患者安全、临床风险意识融入教学设计 |
| **三阶段** | 无 | 院校教育 → 毕业后教育 → 继续教育 |

## 三、本 Skill 的标准遵循声明

本 Skill 在以下方面严格遵循 ISW 国际标准：
- BOPPPS 六步教学设计模型
- 微格教学循环（3轮制）
- 结构化非评价性反馈协议
- 引导员角色定位

在以下方面进行医学教育特色化扩展：
- 多学科适配模板（基础医学/临床医学/护理学/药学）
- 临床案例驱动的教学设计
- 与 OSCE/mini-CEX 等临床评价工具对接
- 医学人文与伦理的自动融入
- 三阶段医学教育（院校/毕业后/继续）适配
