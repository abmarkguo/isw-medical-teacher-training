# 循序问答设计指南

## 一、核心理念

循序问答（Progressive Questioning）是基于Bloom认知层次理论和苏格拉底式提问法，通过由浅入深、层层递进的问题链引导学生完成临床推理和知识建构的教学策略。

在医学教育中，循序问答的核心价值在于：
- **模拟临床推理过程**：从症状→鉴别→检查→诊断→治疗，问题链映射临床思维
- **暴露认知盲区**：通过递进式追问，精准定位学生的知识缺口
- **培养高阶思维**：从记忆理解走向分析评价创造
- **促进主动学习**：每个问题都是学习的驱动力，而非简单的知识检测

## 二、Bloom六层问题设计框架

### 层次结构

| Bloom层次 | 认知动词 | 医学示例 | 问题特征 |
|-----------|---------|---------|---------|
| 1. 记忆 | 说出/列出/识别 | "列出急性心梗的典型症状" | 事实回忆，单一答案 |
| 2. 理解 | 解释/描述/归纳 | "解释为什么心梗患者会有放射痛" | 用自己的话表述机制 |
| 3. 应用 | 完成/执行/计算 | "给这位患者计算TIMI危险评分" | 将知识用于具体情境 |
| 4. 分析 | 比较/区分/分解 | "区分STEMI和NSTEMI的心电图差异" | 拆解成分，找关系 |
| 5. 评价 | 判断/评估/论证 | "评估这位患者的溶栓与PCI选择" | 基于证据做判断 |
| 6. 创造 | 设计/构建/制定 | "为这位患者制定出院后随访方案" | 生成新方案/产品 |

### 设计原则
- 一条问题链通常从Layer 1-2起步，经过Layer 3-4，到达Layer 5-6
- 不是每个链都必须覆盖全部6层，根据目标选择3-5层
- 前一层答案是后一层问题的基础
- 留出"思考等待时间"：Layer 1-2等待3-5秒，Layer 3-4等待10-15秒，Layer 5-6等待30秒+

## 三、医学教育特色问答模式

### 模式一：临床推理阶梯（CRS）
模拟从接诊到决策的完整临床推理过程。

```
第1阶（信息提取）：从病史中提取关键信息
第2阶（模式识别）：这些信息指向哪个/哪些疾病？
第3阶（鉴别推理）：如何区分这些可能性？
第4阶（检查选择）：需要什么检查来确认？
第5阶（结果解读）：检查结果如何解读？
第6阶（决策制定）：最终诊断和治疗方案是什么？
第7阶（反思评价）：如果初始判断错误，后果是什么？
```

### 模式二：机制探查链（MEL）
从临床现象追溯到基础医学机制。

```
第1层（现象观察）：你观察到什么异常？
第2层（关联知识）：这涉及哪个系统/器官？
第3层（机制解释）：为什么会出现这个表现？（病理生理）
第4层（基础追溯）：背后的分子/细胞机制是什么？
第5层（干预推演）：针对这个机制，有哪些干预靶点？
第6层（证据评价）：这些干预的证据等级如何？
```

### 模式三：伦理思辨链（EDL）
从临床情境引发伦理思考。

```
第1层（情境识别）：这个情境中存在什么伦理冲突？
第2层（利益分析）：各方的利益诉求是什么？
第3层（原则调取）：涉及哪些医学伦理原则？
第4层（冲突权衡：这些原则之间如何排序？
第5层（决策论证）：你的决定是什么？理由是什么？
第6层（反思迁移）：如果条件改变，你的决定会改变吗？
```

### 模式四：技能递进链（SKL）
操作技能的分步引导。

```
第1层（准备判断）：操作前需要确认什么？
第2层（步骤排序）：操作的正确顺序是什么？
第3层（关键识别）：哪一步最容易出错？为什么？
第4层（异常处理）：如果遇到XX情况，怎么应对？
第5层（质量评价：如何判断操作是否达标？
第6层（优化建议）：如何提高操作的效率和安全性？
```

## 四、问题链设计流程

### Step 1：确定教学目标
- 从BOPPPS教案的学习目标出发
- 识别目标对应的Bloom层次
- 确定问题链要覆盖的认知层次范围

### Step 2：选择问答模式
根据教学情境选择：
- 病例讨论 → CRS（临床推理阶梯）
- 机制讲解 → MEL（机制探查链）
- 伦理讨论 → EDL（伦理思辨链）
- 技能教学 → SKL（技能递进链）
- 也可组合使用

### Step 3：设计问题链
- 每个问题标注Bloom层次
- 设计"桥接语"连接前后问题
- 为每个问题准备"预期答案"和"追问策略"
- 标注"等待时间"和"互动方式"

### Step 4：设计追问策略
当学生回答不完整或错误时：
- **澄清追问**："你的意思是……？"
- **证据追问**："什么信息让你得出这个结论？"
- **视角追问**："从XX角度再想想？"
- **反例追问**："如果情况是XX，你的答案还成立吗？"
- **桥接追问**："你刚才说了A，那么B呢？"
- **迁移追问**："这和之前学过的XX有什么联系？"

### Step 5：设计互动方式
| 互动方式 | 适用层次 | 操作方法 |
|---------|---------|---------|
| 个人思考 | 所有层次 | 独立思考30秒-1分钟 |
| 同桌讨论 | Layer 3-4 | 两人交换观点 |
| 小组讨论 | Layer 4-6 | 3-5人讨论后汇报 |
| 全班投票 | Layer 1-3 | 选项投票/举手/二维码 |
| 角色扮演 | Layer 5-6 | 模拟医患/团队场景 |
| 白板展示 | Layer 4-6 | 小组在白板画出推理过程 |

## 五、BOPPPS各环节的问答策略

| BOPPPS环节 | 问答目的 | 推荐模式 | Bloom层次 |
|-----------|---------|---------|-----------|
| Bridge-in | 激活先备知识，引发好奇 | CRS第1-2阶 | Layer 1-2 |
| Pre-assessment | 诊断已有水平 | CRS/MEL | Layer 1-3 |
| Participatory Learning | 推进推理，建构新知 | CRS全链/MEL全链/SKL | Layer 3-6 |
| Post-assessment | 检测迁移能力 | CRS变式/MEL | Layer 3-5 |
| Summary | 反思整合 | EDL/反思链 | Layer 5-6 |

## 六、质量评价标准

### 问题链质量自检
- [ ] 问题是否由浅入深、层层递进？
- [ ] 每个问题是否与前一个有逻辑关联？
- [ ] 是否覆盖了目标Bloom层次？
- [ ] 是否有针对错误回答的追问策略？
- [ ] 等待时间是否合理标注？
- [ ] 互动方式是否多样化？
- [ ] 问题是否与临床推理/医学实践直接相关？
- [ ] 最后一个问题是否指向高阶思维？

### 常见问题
| 问题 | 表现 | 修正 |
|------|------|------|
| 层次跳跃 | 从Layer 1直接跳到Layer 5 | 插入中间层次问题 |
| 封闭过多 | 答案只有是/否 | 改为"如何""为什么""什么证据" |
| 缺乏追问 | 学生答错就跳过 | 设计2-3个追问策略 |
| 问题过密 | 一次性抛出多个问题 | 一次一问，等待回答后再追问 |
| 脱离情境 | 纯知识问答无情境 | 每个问题绑定临床情境 |
